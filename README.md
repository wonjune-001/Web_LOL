# League of Legends - Interactive Champion Guide

## 프로젝트 소개

이 프로젝트는 리그 오브 레전드의 포지션별 추천 챔피언 정보를 제공하는 간단한 인터랙티브 웹 가이드입니다. 사용자는 각 포지션(TOP, JGL, MID, BOT, SUP) 메뉴를 통해 챔피언 목록을 확인하고, 챔피언을 클릭하여 해당 챔피언의 장점과 추천 상황을 볼 수 있습니다.

## 주요 기능

- **동적 배경 영상:** 리그 오브 레전드 테마의 동적 배경이 적용되어 있습니다.
- **드롭다운 메뉴:** 각 포지션별 챔피언 목록을 확인할 수 있는 드롭다운 메뉴가 구현되어 있습니다.
- **챔피언 정보 모달:** 챔피언을 클릭하면 해당 챔피언의 장점과 추천 상황이 포함된 상세 정보가 모달 창으로 표시됩니다.
- **반응형 UI:** 마우스 호버 및 클릭에 반응하는 인터랙티브 UI가 적용되어 있습니다.

## 사용 방법

프로젝트 폴더 내의 `index.html` 파일을 웹 브라우저에서 열면 바로 확인할 수 있습니다.
