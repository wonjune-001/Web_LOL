# League of Legends - 매력적인 챔피언 가이드 프로젝트

## 프로젝트 소개

이 프로젝트는 인기 게임 리그 오브 레전드의 챔피언 정보를 포지션별로 안내하는 인터랙티브 웹 가이드입니다. 사용자의 경험을 최우선으로 고려하여 직관적인 탐색과 시각적으로 풍부한 콘텐츠 제공에 중점을 두었습니다. 최신 웹 기술을 활용하여 단순한 정보 나열을 넘어, 사용자에게 몰입감 있는 탐색 경험을 선사합니다.

## 주요 기능

-   **시선을 사로잡는 메인 타이틀 효과:** 페이지 로드 시 메인 타이틀 "리그 오브 레전드"가 마치 타이핑되듯 나타나 사용자에게 강렬한 첫인상을 남깁니다.
-   **매력적인 정적 배경:** 리그 오브 레전드 테마의 고품질 정적 이미지가 배경으로 적용되어 안정적이면서도 게임의 분위기를 효과적으로 전달합니다.
-   **부드러운 페이지 로딩 애니메이션:** 핵심 UI 요소들이 페이지 로드 시 부드럽게 페이드인하며 나타나, 사용자에게 세련되고 깔끔한 시각적 경험을 제공합니다.
-   **직관적인 드롭다운 메뉴:** 각 포지션(TOP, JGL, MID, BOT, SUP)별 챔피언 목록을 편리하게 탐색할 수 있는 드롭다운 메뉴가 개선된 호버 효과와 함께 제공됩니다.
-   **향상된 메뉴 호버 인터랙션:** 내비게이션 및 서브메뉴 항목에 마우스를 올렸을 때 미묘하고 동적인 애니메이션 효과가 적용되어 사용자의 상호작용을 더욱 즐겁게 만듭니다.
-   **챔피언 상세 정보 모달:** 챔피언을 클릭하면 해당 챔피언의 핵심 장점과 활용 팁이 담긴 상세 정보가 모달 창으로 깔끔하게 표시됩니다.
-   **편리한 '맨 위로' 스크롤 버튼:** 페이지를 스크롤할 때 나타나는 버튼을 통해 어느 위치에서든 페이지 상단으로 손쉽게 이동할 수 있습니다.
-   **반응형 UI:** 다양한 기기 및 화면 크기에서도 최적화된 사용자 경험을 제공하도록 설계되었습니다.

## 사용 방법

프로젝트 폴더 내의 `index.html` 파일을 웹 브라우저에서 열면 바로 몰입감 있는 챔피언 가이드를 경험할 수 있습니다.

## 기술 스택

-   HTML5: 웹 페이지 구조 및 콘텐츠 정의
-   CSS3: 매력적인 디자인 및 동적인 UI/UX 구현 (애니메이션, 전환 효과 포함)
-   JavaScript (ES6+): 인터랙티브 기능, DOM 조작 및 동적인 콘텐츠 제어
